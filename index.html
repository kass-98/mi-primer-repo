Anatomia de elemento HTML

Un elemento HTML consta de una etiqueta de apertura, contenido y una etiqueta de cierre. Po rejemplo

<h1>Título principal</h1>

En este caso, <h1> es la etiqueta de apertura, "Título principal" es el contenido y </h1> es la etiqueta de cierre.

Los atributos proporcionan información extra. Ejemplo: class="nota"

HTML Semántico

El HTML semántico utiliza etiquetas que describen claramente su significado y función en el documento. Esto mejora la accesibilidad, la SEO y la mantenibilidad del código.

Ejemplos de etiquetas semánticas

  • <header>: Define el encabezado de una sección o página.
  • <nav>: Define un bloque de navegación.
  • <main>: Define el contenido principal del documento.
  • <article>: Define un artículo independiente.
  • <section>: Define una sección genérica del documento.
  • <footer>: Define el pie de página de una sección o página.
No uses div para todo, elige la etiqueta adecuada, recuerda que el HTML semántico mejora la accesibilidad y SEO.

Contenedores: div vs span

<div> es un contenedor de bloque que se utiliza para agrupar elementos y aplicar estilos o scripts a un bloque completo, es decir, ocupa todo el ancho disponible. Por ejemplo:

<div class="caja">

<span> es un contenedor en línea que se utiliza para agrupar elementos de texto y aplicar estilos o scripts a un fragmento específico, es decir, no ocupa todo el ancho disponible solamente lo que ocupe el elemento . Por ejemplo:

<span class="resaltado">Texto resaltado</span>

Buenas prácticas en HTML

  • Usar etiquetas semánticas para mejorar la accesibilidad y SEO.
  • Incluir atributos alt en las imágenes para describir su contenido.
  • Organizar el contenido de manera clara y estructurada.
  • Evitar el uso excesivo de divs y spans, usar etiquetas adecuadas.
  • Validar el código HTML para asegurar su correcto funcionamiento.
